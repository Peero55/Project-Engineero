# MVP Steering

Target a playable, demoable MVP.

## Prioritize

1. stable startup
2. stable installs and scripts
3. stable Slack flow
4. stable hunt and battle flow
5. stable progression updates
6. one polished battle slice
7. content/admin sanity
8. visual polish

## Avoid Early

- broad refactors
- speculative abstractions
- multiple parallel UI systems
- unnecessary package churn
- redesigning working flows without evidence
