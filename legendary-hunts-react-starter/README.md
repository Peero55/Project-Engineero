# Legendary Hunts React Starter

This repo is a no-drift starter for **Legendary Hunts**: a fantasy-themed learning RPG that uses Slack for daily study rituals and a web app for hunts, battles, mastery tracking, explanations, and admin operations.

## What this repo is
- A React/Next.js starter for the web app
- A dictionary-style handoff for AI agents
- A source-controlled place for prompts, docs, and UI rules

## What this repo is not
- Not the full production game yet
- Not the final schema or Slack app implementation
- Not permission to redesign the product

## Run locally
```bash
npm install
npm run dev
```
