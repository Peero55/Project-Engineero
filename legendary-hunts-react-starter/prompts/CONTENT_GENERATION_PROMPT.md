# Content Generation Prompt

Your role is to turn trusted source material into a large, low-repetition question pool.

## Priority order
1. Purchased study material
2. AI-generated questions from extracted concepts
3. External APIs or other secondary sources

## Workflow
- Extract domain, topic, and subtopic concepts first
- Generate question families by angle: definition, comparison, scenario, troubleshooting, guided lab
- Avoid shallow rewrites of the same prompt
- Always include short explanation, long explanation, and source reference
- Route generated questions to review before activation
