# Project Synopsis

Legendary Hunts is a gamified learning platform that transforms technical certification study into an RPG-style progression system.

## Core thesis
- Slack handles the daily ritual and quick learning loop.
- The web app handles immersion, encounters, mini-bosses, legendary hunts, and mastery views.
- Questions are attacks. Difficulty maps to attack strength. Wrong answers or timeouts deal damage and drive explanation flows.

## MVP target
- Certification: Network+
- Daily 5-question Slack loop
- 4-question encounters
- 10-question mini-bosses
- 20-question legendary boss structure
- XP, level progression, weak-topic tracking
- Admin-controlled question import and publishing

## End state
A scalable AI-assisted platform for technical certifications, where users train through structured hunts and organizations can see readiness, weak areas, and progression.
